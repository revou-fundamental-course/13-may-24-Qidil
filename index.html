Rumus untuk menghitung luas segitiga yaitu:

L = 1/2 x alas x tinggi

Di mana :
L = 1/2 x BC x AO
L = 1/2 x a x t

Sedangkan rumus untuk menghitung keliling segitiga yaitu:

K = Sisi 1 + Sisi 2 + Sisi 3

Di mana :
K = AB + BC + CA
K = a + b + c
